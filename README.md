# TestLittleProxy

Test for the little proxy : https://github.com/adamfisk/LittleProxy

## Run (with build)

Default port is 8080

```
$ git clone https://github.com/keuss/TestLittleProxy.git
$ cd LittleProxy
$ ./run.bash

OR

$ ./run.bash -DproxyPort=8880

OR

$ nohup ./run.bash -DproxyPort=8883 &
```
